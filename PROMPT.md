# AI Chatbot Backend

## **Role:**
You are an expert backend developer specializing in AI-powered chatbot systems with expertise in Node.js, Express.js, database integration, and AI API implementations.

## **Objective:**
Develop a comprehensive AI-Powered Chatbot backend system that integrates with OpenAI/Gemini APIs, manages user conversations, stores chat history, and provides admin functionality for monitoring and analytics.

## **Context:**
This project involves building a scalable backend service for an AI chatbot that can handle multiple users, maintain conversation history, integrate with AI services, and provide administrative oversight through query logs and analytics.

## **Instructions:**

### **Instruction 1:**
Implement OpenAI/Gemini API integration with proper error handling, rate limiting, and response processing for natural language conversations.

### **Instruction 2:**
Design and implement a robust database schema to store user queries, chatbot responses, conversation threads, and user sessions with proper indexing and relationships.

### **Instruction 3:**
Create RESTful APIs for user chat history retrieval, conversation management, and real-time chat functionality with proper authentication and authorization.

### **Instruction 4:**
Develop an admin panel backend with APIs for query logs, user analytics, conversation monitoring, and system health metrics.

## **Notes:**
• Note 1: Ensure proper API key management and security for AI service integrations
• Note 2: Implement efficient database queries and caching for chat history retrieval
• Note 3: Add comprehensive error handling and logging for debugging and monitoring
• Note 4: Design scalable architecture to handle multiple concurrent users
• Note 5: Include proper input validation and sanitization for security
